Excel Custom Number Formatting

Complete guide to every symbol

Sarat Rout February 17, 2026 9 min read Excel Tips

Excel's custom number formatting is one of its most powerful yet underutilized features. With a handful of symbols, you can control exactly how numbers, dates, text, and even units appear‚Äîwithout changing the underlying value. This guide explains every symbol you'll encounter, with practical examples you can use immediately.

Think of custom formats as a language: each symbol tells Excel how to display your data. Once you learn the grammar, you can create formats that make your spreadsheets clearer, more professional, and easier to read.

üî£ Literal Characters & Text

\

Text placeholder. Displays text exactly as entered‚Äîuseful when you want to combine text formatting with conditions.

Space filler. Adds space equal to the width of the next character‚Äîexcellent for alignment in columns.

Thousand separator. Also scales numbers‚Äîone comma divides by 1000, two commas by 1,000,000.

Repeats the following character to fill the remaining cell width‚Äîgreat for creating visual separators.

%

üìù Practical Format Examples

Currency Format: $#,##0.00 ‚Üí 1234.5 displays as $1,234.50

Units Format: 0.0 "kg" ‚Üí 75.25 displays as 75.3 kg

Phone numbers Format: (###) ###-#### ‚Üí 2125551234 displays as (212) 555-1234

Scaling Format: #,##0,,"M" ‚Üí 2500000 displays as 2.5M

Fractions Format: # ??/?? ‚Üí 0.333 displays as 1/3

Dates with text Format: yyyy "Date" mm/dd ‚Üí Feb 17, 2026 displays as 2026 Date 02/17

üî¢ Four-Section Format Structure

Custom formats can have up to four sections separated by semicolons. Each section controls a different type of value:

[Positive]; [Negative]; [Zero]; [Text]
SectionControlsExample
1stPositive numbers0.00 ‚Üí 5 shows as 5.00
2ndNegative numbers[Red]-0.00 ‚Üí -5 shows as red -5.00
3rdZero values"Zero" ‚Üí 0 shows as Zero
4thText"Text: "@ ‚Üí Hello shows as Text: Hello

Complete example: 0.00;[Red]-0.00;"Zero";"Text: "@ handles all four value types with different formatting.

Pro tips:

  • Use 0 when you need consistent decimal places or leading zeros.
  • Use # when you want numbers to appear as naturally typed.
  • Use ? to align decimal points in a column‚Äîadds spaces instead of zeros.
  • Remember: formatting only changes display‚Äîthe actual cell value remains unchanged for calculations.
  • To display both currency symbol and text, combine: $#,##0.00 "USD"
