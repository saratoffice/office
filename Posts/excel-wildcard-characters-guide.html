1. What Are Wildcard Characters?

Excel Wildcards — The Pattern Matching Toolkit

A wildcard is a special character that substitutes for one or more unknown characters in a text search. Think of it as a "joker" — it can represent any value in the position where it appears. According to Microsoft Support and Exceljet, Excel supports exactly three wildcard characters — and all three work together to give you flexible, powerful pattern matching across formulas, filters, and Find & Replace.

Wildcards are active by default in Excel's Find & Replace dialog and are supported in a specific set of functions. They are not case-sensitive — searching for apple* will match "apple", "Apple", and "APPLE" alike.

* Any number of characters ? Exactly one character ~ Escape — treat * or ? as literal
Asterisk (*) wildcard used in a COUNTIF formula — counts cells matching a partial text pattern
VLOOKUP with asterisk wildcard — perform partial-match lookups when you only know part of the entry

3. Asterisk (*) — Any Number of Characters

*

Asterisk (*)

Matches any number of characters — including zero characters

The asterisk is the most powerful and flexible wildcard. It represents zero or more characters in any position. Place it at the start, end, or both sides of your search text to match different patterns. Because it matches zero characters, Lemon* also matches the word "Lemon" itself — not just words starting with "Lemon".

Pattern: *tion — matches any text ending with "tion"
action caution education station tion tional
Pattern: *apple* — matches any text containing "apple" anywhere
apple green apple pineapple apple juice aple
Pattern: L*n — starts with L, ends with n, anything between
Lemon Loan Learn Ln Lemon juice

4. Question Mark (?) — Exactly One Character

?

Question Mark (?)

Matches exactly one character — any single character

The question mark is more precise than the asterisk — it matches exactly one character, no more, no less. Each ? in your pattern accounts for exactly one character position. Use it when you know the structure of the text but need to account for variation in specific positions — such as unknown middle initials, single-digit codes, or spelling variants.

Pattern: b?g — three-letter words starting with b, ending with g
bag beg big bog bug bg (0 between) baag (2 between)
Pattern: pri?e — five-letter words matching the structure
price pride prize prise (but prise — still matches actually)
Pattern: ??? — any text that is exactly three characters long
cat dog car ABC at (2 chars) cats (4 chars)
Pattern: ???-??? — two groups of exactly 3 characters separated by a hyphen
ABC-DEF 123-456 IN-001 (no — 2 before hyphen)

5. Tilde (~) — Escape Character for Literal Wildcards

~

Tilde (~)

Escape character — cancels the wildcard meaning of * or ?

The tilde is the "anti-wildcard." It is placed immediately before an asterisk, question mark, or another tilde to tell Excel to treat that next character as a literal character — not a wildcard. This is essential when your actual data contains asterisks or question marks (e.g., product codes, invoice numbers, formula strings).

Tilde escape sequences
Pattern Searches For Example Match
~* A literal asterisk character 5*3 = 15
~? A literal question mark character How are you?
~~ A literal tilde character ~discount
Pattern: pivot~* — finds "pivot*" literally, not "pivot table"
pivot* pivot table pivoting
Pattern: Aus~* — finds "Aus*" literally
Aus* Australia Austria

6. Combining Wildcards — Advanced Patterns

Wildcards can be freely combined with each other and with regular text to create precise, powerful search patterns:

*e?n*
Contains 'e', then any one character, then 'n', with anything before or after
Matches: learn, screen, even, open
J* Smith
Starts with J (first name), then a space, then ends with Smith
Matches: John Smith, Jane Smith, J. Smith
*@*.com
Anything before @, domain, ends in .com — loose email pattern
Matches: user@gmail.com, a@b.com
IN-???
Starts with IN-, followed by exactly three characters
Matches: IN-001, IN-ABC, IN-123
????-????
Two groups of exactly four characters separated by a hyphen
Matches: ABCD-1234, TEST-CODE
*~**
Contains a literal asterisk anywhere in the text
Matches: 5*3, price*, note*

7. Excel Functions That Support Wildcards

Not all Excel functions accept wildcard characters. The following table lists all functions that officially support wildcards, along with those that do not — which is a common source of confusion:

Function Wildcards? Example Formula What It Does
COUNTIF / COUNTIFS ✓ Yes =COUNTIF(A:A,"*tion") Counts cells where text ends with "tion"
SUMIF / SUMIFS ✓ Yes =SUMIF(A:A,"*apple*",B:B) Sums col B where col A contains "apple"
AVERAGEIF / AVERAGEIFS ✓ Yes =AVERAGEIF(A:A,"Pro*",B:B) Averages col B where col A starts with "Pro"
VLOOKUP ✓ Yes =VLOOKUP("*"&D1&"*",A:B,2,0) Partial-match lookup using cell D1 as search term
HLOOKUP ✓ Yes =HLOOKUP("Jan*",A1:Z2,2,0) Horizontal lookup matching text starting with "Jan"
XLOOKUP ✓ Yes (match_mode=2) =XLOOKUP("*Adams",A:A,C:C,"Not found",2) Finds any entry ending with "Adams"; match_mode 2 enables wildcards
MATCH ✓ Yes =MATCH("b?g",A:A,0) Returns position of first match for pattern
XMATCH ✓ Yes (match_mode=2) =XMATCH("*Smith",A:A,2) Modern MATCH — supports wildcard matching with match_mode=2
SEARCH ✓ Yes =SEARCH("ap*le",A1) Case-insensitive; returns position of pattern match
MAXIFS / MINIFS ✓ Yes =MAXIFS(B:B,A:A,"Pro*") Max/Min of col B where col A matches pattern
IF ✗ No =IF(COUNTIF(A1,"Amy*"),"Yes","No") IF doesn't support wildcards — use COUNTIF as a workaround
FIND ✗ No Use SEARCH instead FIND is case-sensitive and does not support wildcards at all
XLOOKUP and XMATCH note: Unlike VLOOKUP and MATCH — which automatically recognise wildcards — XLOOKUP and XMATCH require you to explicitly set match_mode = 2 (wildcard match) in the formula. Without this, wildcards are treated as literal characters.

8. Real Formula Examples

Task Formula How It Works
Count cells ending with "tion" =COUNTIF(A:A,"*tion") * matches any characters before "tion"
Count cells that are exactly 7 chars =COUNTIF(A:A,"???????") Seven ? marks — each matches exactly one character
Count non-blank text cells =COUNTIF(A:A,"*") * matches any text; blank cells and numbers are excluded
Sum sales for products starting with "Pro" =SUMIF(A:A,"Pro*",B:B) Matches "Product A", "Pro-Series", "Pro Max" etc.
Lookup by partial name from cell D1 =VLOOKUP("*"&D1&"*",A:B,2,0) Ampersand joins wildcards with cell value for flexible lookup
XLOOKUP — find entry ending with "Adams" =XLOOKUP("*Adams",A:A,C:C,"Not found",2) match_mode=2 enables wildcard matching in XLOOKUP
IF with wildcard (workaround) =IF(COUNTIF(A1,"Amy*"),"Yes","No") COUNTIF returns 1 (true) or 0 (false) — IF acts on that result
Find literal asterisk in data =COUNTIF(A:A,"*~**") ~* tells Excel the middle * is literal; outer * match surrounding text
Match position of pattern =MATCH("b?g",A:A,0) Returns row position of first match for bag/beg/big/bog/bug
Average of entries starting with "Pf" =AVERAGEIF(A:A,"Pf*",B:B) Matches Pfizer, Pfaff, etc. and averages corresponding B values

9. Where Else to Use Wildcards

Filter Drop-down

Type wildcard patterns in the column filter search box — e.g. A* shows only entries starting with A.

Find & Replace

Wildcards are active by default in Find & Replace (Ctrl+H). jo* finds Jon, John, Joanna, Joshua simultaneously.

Conditional Formatting

Use =COUNTIF(A1,"G*") as a formula rule to highlight all cells starting with "G" automatically.

PivotTable Filters

Wildcard patterns work in label filters inside PivotTables — filter fields by partial text matches.

10. Limitations — What Wildcards Cannot Do

Wildcards only work with text — not numbers. Excel stores numbers in a binary format optimised for calculations, not as text strings. If you try =COUNTIF(A:A,"4*") on a column of numbers, it will return 0. To apply wildcards to numbers, first convert them using the TEXT() function or by formatting cells as text, then use your wildcard formula.
LimitationDetailsWorkaround
Numbers Wildcards don't match numeric cells — only text Convert to text: =COUNTIF(TEXT(A:A,"0"),"4*") or use SEARCH with ISNUMBER
IF function IF() does not directly support wildcards Use =IF(COUNTIF(A1,"Amy*"),"Yes","No")
FIND function FIND is case-sensitive and does not support wildcards Use SEARCH instead — it is case-insensitive and supports wildcards
XLOOKUP / XMATCH Wildcards are not enabled by default in these functions Set match_mode argument to 2 explicitly
False positives *Smith* also matches "Smithson", "Blacksmith", "Goldsmith" Use more specific patterns: e.g. " Smith" with a leading space if it's a last name
Dates Wildcards don't match date-formatted cells directly Use TEXT() to convert dates to strings first, then apply wildcards

11. Pro Tips and Common Mistakes

Pro Tips

  • Wildcards are not case-sensitive. *apple* matches "apple", "Apple", "APPLE" — no need for separate patterns.
  • Combine with ampersand (&) for dynamic patterns. "*"&A1&"*" builds a wildcard pattern from a cell value — use in VLOOKUP, COUNTIF, SUMIF.
  • Count all text cells: =COUNTIF(A:A,"*") counts every non-blank text cell (ignores numbers and blanks).
  • Escape when your data contains wildcards. If a product code is "Part*A", use Part~*A to find it literally.
  • To find a literal tilde: Use ~~ — two tildes mean "a literal tilde character."
  • SEARCH supports wildcards; FIND does not. Always use SEARCH for pattern-based partial text checks.
  • Design patterns carefully. Broad patterns like *A* may match far more cells than intended — be as specific as the data allows.

Quick Cheat Sheet — Excel Wildcards at a Glance

*

Any number of chars (including zero) — most flexible wildcard

?

Exactly one character — use for fixed-length patterns and code formats

~

Escape — places before * or ? to treat them as literal characters

~~

Searches for a literal tilde (~) in the data

✓

Supported by: COUNTIF/S, SUMIF/S, AVERAGEIF/S, VLOOKUP, HLOOKUP, XLOOKUP (mode 2), MATCH, XMATCH (mode 2), SEARCH, MAXIFS, MINIFS

✗

Not supported by: IF, FIND, EXACT, or any formula operating directly on numbers

In summary: Excel's three wildcard characters — the asterisk (*), question mark (?), and tilde (~) — are among the most useful tools in any Excel user's toolkit. Whether you're counting entries that follow a pattern, summing values by partial category, performing partial-match lookups with VLOOKUP or XLOOKUP, or cleaning inconsistent data with Find & Replace, mastering wildcards will make your work significantly faster and more flexible. The key things to remember: wildcards only work with text (not numbers directly), the IF function needs a COUNTIF workaround, XLOOKUP and XMATCH require match_mode = 2, and the tilde is your escape hatch whenever your actual data contains wildcard characters.
