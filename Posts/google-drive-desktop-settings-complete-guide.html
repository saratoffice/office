Google Drive for Desktop — What It Does

Google Drive for Desktop is a native application for Windows and macOS that brings your entire Google Drive — including My Drive, Shared Drives, and folders from other computers — directly into your file system. It eliminates the need to work through a browser, gives you full offline capability, and lets you back up local folders and photos automatically to the cloud.

The app supports up to 4 Google accounts simultaneously, offers two powerful sync modes (Stream and Mirror), and provides granular control over bandwidth, cache, proxy, Google Photos uploads, and more through its Preferences → Advanced Settings menu.

1. Stream Files vs Mirror Files — The Core Sync Choice

The most fundamental setting in Google Drive for Desktop is how your My Drive folder syncs with your computer. You choose between two modes — Stream files and Mirror files. To switch: open the app → Settings → Preferences → left panel "Folders from Drive" → "My Drive syncing options."

Stream Files (Recommended for most users)

Files are stored in the cloud and only downloaded temporarily when you open or use them. You can see all files in File Explorer/Finder without downloading them — like Netflix for your files.

Best for: Saving disk space; users with a reliable internet connection who don't need all files offline constantly.

Offline access: Right-click any file or folder → "Offline access" → "Available offline" to pin it locally.

G:\ → My Drive (files visible but cloud-only until opened)

Mirror Files

Files are stored both in the cloud and on your computer — a complete local copy of My Drive is always on your device. Changes sync in both directions automatically.

Best for: Users who need all files available offline at all times; slower or unreliable internet connections; maximum redundancy.

Trade-off: Uses significantly more local disk space — the size of your entire My Drive is downloaded to your computer.

C:\Users\Name\Google Drive\My Drive (full local copy)
Important before switching modes: Always ensure your recent changes have finished syncing before switching between Stream and Mirror modes. If you switch from Mirror to Stream and delete the old local folder before all changes have uploaded, those un-synced files will be permanently lost. Shared drives always use streaming; folders synced from your computer always use mirroring.

2. Real-Time Presence in Microsoft Office

Real-Time Presence in Microsoft Office

See who is currently editing a shared Office file stored in Google Drive

Collaboration Feature

What It Is: When you have a Microsoft Word, Excel, or PowerPoint file stored in Google Drive for Desktop and open it, real-time presence lets you see if someone else is simultaneously editing the same file. You'll see a coloured indicator — a dot, cursor, or the other person's name — showing live activity in the document.

When It's On (Default)

  • Live indicator shows in the Office application when a colleague is in the file
  • Prevents accidental overwrites in shared files
  • Essential for team collaboration on Word/Excel/PowerPoint stored in Drive

When Turned Off

  • Others will not see that person's presence indicator — they appear invisible to the real-time view
  • You still edit the file but without the live collaboration indicator showing others you're there
macOS users: Before using real-time presence on macOS, you must grant Google Drive accessibility permission. Go to System Settings → Privacy & Security → Accessibility → enable Google Drive.

3. Cache Settings & Local Cached Files Directory

Cache Settings & Local Cached Files Directory

Control where and how much space temporary files use on your computer

Storage Management

What It Is: When you stream files, Google Drive stores recently accessed file data in a local cache on your hard drive — enabling faster re-opening without re-downloading from the cloud. The cache directory setting lets you control where this cache lives and how large it can grow. This is found at: Preferences → Advanced Settings → "Local cached files directory."

Key Options

  • Limit Cache Size: Set a maximum MB/GB cap so the cache doesn't grow unchecked
  • Change Cache Location: Click "Change" to move the cache folder to a different drive or path
  • Offline Files: Files you've pinned for offline access are not affected by cache size limits — they are stored separately

Default Cache Location

Windows:

C:\Users\<Username>\AppData\Local\Google\DriveFS

macOS: Managed by macOS File Provider — cache location cannot be changed on macOS using File Provider.

Moving Cache to Another Drive (Windows)

Example: Your C: drive is almost full but you have ample space on D: drive — change the cache location to D:\GoogleDriveCache to free up system drive space.

  • No data loss — only temporary cached files are moved, not your actual Drive files
  • Performance may be affected on slower drives (e.g., USB drives or older HDDs)
  • Ensure the new location has sufficient free space and is a reliable, always-connected drive
  • If the cache directory path becomes unavailable, you cannot use Google Drive for Desktop until it is restored

4. Google Drive Streaming Location

Google Drive Streaming Location

Choose how and where Google Drive appears on your computer's file system

Location Setup

What It Is: This setting controls how your Google Drive files appear on your computer when using Stream mode — either as a virtual drive letter or as a regular folder on your existing drives. Found at: Preferences → Advanced Settings → "Google Drive streaming location."

Drive Letter (Windows)

Maps your Google Drive to a virtual drive letter (e.g., G:\ or H:\) in Windows File Explorer. Google Drive behaves exactly like a physical hard drive — you can navigate, drag files, and use it in file-open dialogs from any application.

You can change the letter to any available letter using the "Drive letter" drop-down in preferences.

G:\My Drive (or any available letter)

Folder (Windows & macOS)

Instead of a drive letter, maps Google Drive to a specific folder on your existing drive — for example, inside your home directory or a dedicated folder on another drive.

On macOS, the streaming location is typically managed by macOS (File Provider) — you may see "Folder location is controlled by macOS" and cannot change it.

D:\MyGoogleDrive\ (custom path)
Note on Mirror mode: When mirroring, you select a regular folder on your computer during setup — that folder stores a full copy of My Drive. The streaming location setting primarily applies to Stream mode.

5. Show Google Drive in Quick Access (File Explorer)

Show Google Drive in Quick Access

Pin Google Drive to the sidebar of Windows File Explorer for one-click access

What It Is: When enabled, Google Drive appears in the "Quick Access" section of Windows File Explorer's left-hand sidebar — alongside pinned folders like Desktop, Documents, and Downloads. This makes it instantly accessible from any File Explorer window without navigating to the drive letter each time.

Practical benefit: Google Drive behaves exactly like a local folder when pinned to Quick Access — drag and drop files in and out, open files directly, or save documents to it from any application's Save dialog, just as you would with any other local drive or folder.

6. Google Photos — Upload Size & File Types

Google Photos Settings

Control how photos and videos from your computer back up to Google Photos

What It Is: Google Drive for Desktop can automatically back up photos and videos from your computer to Google Photos. You control the upload quality and which file types are included. These settings are under: Preferences → Advanced Settings → Google Photos file types.

Upload Quality

Storage Saver

Photos are compressed and reduced in quality to save Google account storage. Files above 16 MP for photos or 1080p for videos are compressed. Items stored in Storage Saver quality no longer count against your quota (varies by Google account terms).

Original Quality

Photos and videos are uploaded at full, uncompressed resolution. They count against your Google account storage quota (shared across Drive, Gmail, and Photos).

File Type Options

Under Advanced Settings, you can check or uncheck:

  • Upload Screenshots: Automatically back up screenshots captured on your computer to Google Photos. Disable if you don't want screenshots in your photo library.
  • Upload RAW Files: Back up RAW image files (e.g., CR2, NEF, ARW) from professional cameras. Enable this if you shoot in RAW format and want full-resolution originals backed up.
Tip: Enable RAW upload if you're a photographer — RAW files are not lossy-compressed and represent your original capture.
Pausing sync stops Google Photos backups along with all other Drive sync activity (file streaming and mirroring). Resume sync to restart photo uploads.

7. Launch on Login, Proxy & Bandwidth Settings

Launch on Login

Google Drive for Desktop starts automatically every time you log in to your computer, ensuring your files are always syncing in the background without manual intervention.

To disable: Preferences → Advanced Settings → uncheck "Launch Google Drive when you login to your computer."

Recommended to keep enabled for automatic sync.

Proxy Settings

Controls how Google Drive connects to the internet when your network uses a proxy server.

Auto-detect (default): Uses the proxy settings configured in your operating system — suitable for most users and corporate environments.

Direct connection: Bypasses all proxy settings and connects directly to Google's servers — use if you're on a network without a proxy and auto-detect causes connection issues.

Found at: Preferences → Advanced Settings → Proxy settings.

Bandwidth Settings

Set maximum upload and download speed limits for Google Drive sync activity in kilobytes per second (KB/s) or megabytes per second (MB/s).

Example: Set a 2,000 KB/s (≈2 MB/s) upload limit so large file uploads don't saturate your internet connection during video calls or other high-bandwidth activities.

Found at: Preferences → Advanced Settings → Bandwidth settings.

8. Hotkeys, Diagnostics & Notification Preferences

Configure Hotkeys

Set a global keyboard shortcut to trigger Google Drive's enhanced search function — letting you search all your streamed Drive files directly from the app, unlike Windows Search or macOS Spotlight which can't index cloud-only files.

Default (Windows): Ctrl + Alt + G

You can customise the key combination to any shortcut that doesn't conflict with other applications.

Help Improve Google Drive

When enabled, Google Drive sends anonymous diagnostic data and usage statistics to Google — helping improve the app's performance and stability.

Enable if you're comfortable contributing usage data. Disable if you prefer maximum privacy or are in a sensitive data environment.

This setting has no effect on your files or sync behaviour.

Notification Preferences

Control which system notifications Google Drive for Desktop shows.

Prompt me to back up devices: Receive reminders to back up connected devices — phones, cameras, or USB drives — to Google Photos when they are plugged into your computer.

Important sync errors and time-sensitive alerts appear in your system notifications panel regardless of this setting.

9. My Drive vs Other Computers — Understanding the Folders

When you open Google Drive for Desktop in File Explorer or Finder, you see two primary folder views alongside Shared drives. Each represents a different aspect of your Drive account. (Note: "Other computers" only appears if Google Drive is installed and syncing on at least one other device linked to your account.)

10. How to Use Both Folders Together

Syncing Files from Other Computers

Install Google Drive for Desktop on your other computers and configure which local folders (Desktop, Documents, Downloads) to sync to Drive. Those folders automatically appear under "Computers" → [Device Name] in your Drive. You can then access them from any device — including through drive.google.com.

Moving Files to My Drive

To consolidate files from "Other Computers" into your primary workspace, simply drag them from the other computer's folder into My Drive — either in File Explorer or on drive.google.com. Once in My Drive, they're fully two-way synced and accessible everywhere, regardless of whether the originating computer is online.

Organising Across Devices

Create a structured folder hierarchy in My Drive (e.g., Work/Projects/, Personal/Photos/, Archive/) and move files from "Other Computers" into the appropriate folders. This gives you a clean, device-agnostic organisation system visible from all your computers, mobile devices, and the web.

In Conclusion

My Drive

The primary workspace and storage hub for all your files on Google Drive. Everything you upload, create, or move here syncs two-way across all devices — computers, phones, and the web. Use My Drive for all files you actively work with and want universally accessible.

Other Computers

A live view of local folders synced from your other devices. It's your bridge for accessing files left on another machine and for backing up local folders like Desktop or Documents to the cloud — without manually moving them to My Drive.

By mastering Stream vs Mirror, configuring cache to avoid disk space issues, pinning Drive to Quick Access, setting bandwidth limits to avoid network saturation, and understanding the My Drive vs Computers folder structure — you can make Google Drive for Desktop work seamlessly as a core part of your daily productivity workflow.
